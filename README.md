## ⚠️ PROJECT ARCHIVED
This project is no longer under active development and is preserved here for historical purposes only. This project was an old fork of MultiCraft's official [GitHub](https://github.com/MultiCraft/MultiCraft).

## ⛏️ MultiCraft Game (Archived)
This was the default game bundled with the MultiCraft engine.

For historical information, you can still check the [original forum topic](https://forum.MultiCraft.net/viewtopic.php?f=15&t=9724) or the [MultiCraft Wiki](https://wiki.MultiCraft.net/Subgames/MultiCraft_Game) for more information.

## ℹ️ Status Notice
Active Development: Stopped.

Support: No longer provided.

Successor: Users are encouraged to check MultiCraft's official [GitHub](https://github.com/MultiCraft/MultiCraft) for the latest default game implementations.

## ⬇️ Installation (Legacy)
Unzip the archive, rename the folder to 'MultiCraft_game' and place it in '../MultiCraft/games/'

GNU/Linux: If you use a system-wide installation, place it in '~/.MultiCraft/games/'.

The MultiCraft engine can be found at [GitHub](https://github.com/MultiCraft/MultiCraft).

## 🏛️ Compatibility
Historically, the MultiCraft Game master HEAD was kept compatible with the master HEAD of the MultiCraft engine. When stable releases were made, MultiCraft Game and the MultiCraft engine were packaged together at [www.MultiCraft.net](https://www.MultiCraft.net/downloads).
